# 🎮 Simon Says Game

A modern Simon Says memory game built using HTML, CSS, and JavaScript.

## 🚀 Features

* 🎯 Increasing difficulty levels
* 🏆 Leaderboard (Firebase)
* 🔥 Daily streak system
* 👥 Players count
* 📱 Fully responsive (mobile-friendly)
* 🔊 Sound effects

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Firebase

## 🎮 How to Play

1. Click "Start Game"
2. Watch the color sequence
3. Repeat the sequence correctly
4. Game gets harder every level!
